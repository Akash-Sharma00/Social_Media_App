# social_media

A new Flutter App for post text/images or both. And interact with world!

## Developed with
- Flutter 
- Firebase
- Mongo DB
- NodesJs
- ZegoCloud

## State Management
- Block

## Feature
- Post (text/images)
- Chatting
- Like 
- Comment
- Video Call
